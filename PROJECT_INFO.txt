HOW TO USE
================================================================================

1. INSTALL DEPENDENCIES:
   cd production_line_barcode_verifier
   pip install -r requirements.txt

2. RUN THE SYSTEM:
   python production_line_verifier.py

3. OPERATE:
   - Press 'C' to capture reference barcode
   - Press 'S' to start production mode
   - Press 'Q' to quit

================================================================================
KEYBOARD CONTROLS (LAPTOP MODE)
================================================================================

C = Capture reference barcode (from correct product)
S = Start/Stop production verification
R = Reset reference barcode
L = View recent logs
H = Show help
Q = Quit system

================================================================================
DETECTION STATES
================================================================================

✅ PASS (Correct Product)
   - Green box around barcode
   - Short high beep
   - Console: [PASS]
   - Action: Continue production

❌ MISMATCH (Wrong Product)
   - Red box around barcode
   - Double beep sound
   - Console: [ALERT] BARCODE MISMATCH
   - Action: Remove/replace product

⚠️ NO BARCODE (Missing/Damaged)
   - Alert message
   - Long low beep
   - Console: [ALERT] NO BARCODE DETECTED
   - Action: Check product quality
