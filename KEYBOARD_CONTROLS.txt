================================================================================
PRODUCTION LINE BARCODE VERIFIER - KEYBOARD CONTROLS
================================================================================

🎮 INTERACTIVE CONTROLS (Press these keys while the system is running):

┌─────────────────────────────────────────────────────────────────────────────┐
│  C  - CAPTURE REFERENCE BARCODE                                            │
│      • Point camera at correct product barcode                             │
│      • Press 'C' to set as reference                                       │
│      • You'll hear 3 ascending beeps when successful                       │
│      • MUST do this before starting production!                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  S  - START/STOP PRODUCTION MODE                                           │
│      • Press 'S' to start automatic verification                           │
│      • System will scan every 1.5 seconds                                  │
│      • Press 'S' again to pause                                            │
│      • Requires reference barcode to be set first                          │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  R  - RESET REFERENCE BARCODE                                              │
│      • Clears the current reference barcode                                │
│      • Stops production mode                                               │
│      • Use when switching to different products                            │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  L  - VIEW RECENT LOGS                                                     │
│      • Shows last 20 scan results                                          │
│      • Displays in terminal window                                         │
│      • All results also saved to CSV file                                  │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  H  - SHOW HELP                                                            │
│      • Displays this help information                                      │
│      • Shows barcode detection tips                                        │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  Q  - QUIT SYSTEM                                                          │
│      • Safely shuts down the system                                        │
│      • Shows final statistics                                              │
│      • Saves all logs to CSV file                                          │
└─────────────────────────────────────────────────────────────────────────────┘

================================================================================
WORKFLOW EXAMPLE:
================================================================================

1. Start system: python3 production_line_verifier_BARCODE.py
2. Point camera at correct product barcode
3. Press 'C' → Reference captured! (3 beeps)
4. Press 'S' → Production mode started
5. Show products to camera:
   • Same barcode → Green box + short beep (PASS)
   • Different barcode → Red box + double beep (MISMATCH)
   • No barcode → Long beep (NO_BARCODE)
6. Press 'S' → Pause production
7. Press 'Q' → Quit and view statistics

================================================================================
AUDIO ALERTS:
================================================================================

🔊 Reference Captured: Beep-Beep-Beep (ascending tones)
🔊 Pass/Success:       Beep (short, high pitch)
🔊 Mismatch:           Beep-Beep (medium pitch, double)
🔊 No Barcode:         Beeeeeep (long, low pitch)

================================================================================
VISUAL FEEDBACK:
================================================================================

🟢 Green Box:     Correct barcode (PASS)
🔴 Red Box:       Wrong barcode (MISMATCH)
⚠️  Alert Message: No barcode detected

================================================================================
TIPS FOR BEST RESULTS:
================================================================================

✅ Good lighting - Ensure barcode is well-lit
✅ Steady hands - Hold phone/camera steady
✅ Right distance - Keep 15-30 cm from camera
✅ Clean barcode - No damage or dirt
✅ Flat surface - Don't curve the barcode
✅ Try angles - Slight rotation can help
