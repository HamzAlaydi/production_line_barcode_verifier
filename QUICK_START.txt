================================================================================
STEP-BY-STEP USAGE
================================================================================

STEP 1: CAPTURE REFERENCE BARCODE
----------------------------------
1. Start the program
2. Place a CORRECT product with valid barcode in front of camera
3. Press 'C' key
4. Wait for sound confirmation (3 ascending beeps)
5. You should see: "REFERENCE BARCODE CAPTURED!"

STEP 2: START PRODUCTION VERIFICATION
--------------------------------------
1. Press 'S' key to start production mode
2. System will now scan products automatically every 1.5 seconds
3. Watch for results:
   
   [PASS] - Green box, short beep
   - Barcode matches reference
   - Product is correct
   
   [MISMATCH] - Red box, double beep  
   - Wrong barcode detected
   - Remove/replace product
   
   [NO BARCODE] - Long low beep
   - No barcode found
   - Product may be damaged or empty

STEP 3: MONITOR PRODUCTION
---------------------------
- Statistics shown on screen in real-time
- All results logged to: production_log.csv
- Press 'S' again to pause production mode
- Press 'L' to view recent logs
- Press 'Q' to quit

================================================================================
KEYBOARD CONTROLS (Laptop Testing)
================================================================================

C - Capture reference barcode (MUST DO THIS FIRST!)
S - Start/Stop production verification  
R - Reset reference barcode
L - View recent logs
H - Show help
Q - Quit system

AUDIO ALERTS
================================================================================

Reference Captured: Beep-Beep-Beep (ascending)
Pass/Success:       Beep (short, high pitch)
Mismatch:           Beep-Beep (medium pitch)
No Barcode:         Beeeeeep (long, low pitch)

================================================================================
TROUBLESHOOTING
================================================================================

Problem: "No reference barcode set"
Solution: Press 'C' first to capture reference before starting production

Problem: Barcode not detected
Solution: 
- Ensure good lighting
- Hold barcode steady
- Make sure barcode is in focus
- Clean camera lens
- Try different angle/distance

Problem: No audio
Solution: Check system volume and speakers

Problem: Camera error
Solution: 
- Close other apps using camera
- Reconnect camera
- Check camera drivers
